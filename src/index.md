# About

I am an experienced front-end developer who enjoys building modern UI using the latest in web technologies. I frequently leverage the <a href="https://reactjs.org/" target="_blank">React</a> and <a href="https://redux.js.org/" target="_blank">Redux</a> ecosystems (<a href="https://redux-toolkit.js.org/" target="_blank">Redux Toolkit</a>). I ❤️ <a href="https://styled-components.com/" target="_blank">styled-components</a>, <a href="https://xstyled.dev/" target="_blank">xstyled</a>, <a href="https://tailwindcss.com/docs/installation" target="_blank">Tailwind CSS</a>.

Currently, I work at <a href="https://www.kaspersky.com/?ignoreredirects=true" target="_blank">Kaspersky</a> on the Infrastructure Solutions & Services Development team.

I started my journey in Computer Science at the <a href="https://www.miit.ru/en" target="_blank">Russian University of Transport (MIIT)</a> in Moscow. The first programming languages that I learned were <a href="https://en.wikipedia.org/wiki/Turbo_Pascal" target="_blank">Turbo Pascal</a> and <a href="https://en.wikipedia.org/wiki/Assembler" target="_blank">Assembler</a>. After graduation, I started working for a small company that created training programs for train drivers. We've used various technologies: <a href="https://en.wikipedia.org/wiki/Adobe_Flash" target="_blank">Adobe Flash</a>, <a href="https://www.ogre3d.org/" target="_blank">Ogre-3D</a>, <a href="https://en.wikipedia.org/wiki/Augmented_reality" target="_blank">AR</a> (when it first came into being). Next I started exploring .NET technologies such as  <a href="https://dotnet.microsoft.com/en-us/apps/aspnet" target="_blank">ASP.NET</a>, <a href="https://en.wikipedia.org/wiki/Windows_Forms" target="_blank">Windows Forms</a>, <a href="https://en.wikipedia.org/wiki/Windows_Presentation_Foundation" target="_blank">WPF</a>, <a href="https://en.wikipedia.org/wiki/Microsoft_SQL_Server" target="_blank">Microsoft SQL Server</a>, etc. I worked as a back-end developer for several years. Then I started to get interested in the front-end. At that time, the popular framework <a href="https://angularjs.org/" target="_blank">AngularJS</a> appeared, which blew everyone's minds. I decided to become a full-fledged front-end developer. Then came <a href="https://reactjs.org/" target="_blank">React</a>. Since then it has been my go-to tool. Also, I play with different things like <a href="https://effector.dev/" target="_blank">effector</a>, <a href="https://xstate.js.org/" target="_blank">XState</a>, <a href="https://vuejs.org/" target="_blank">Vue.js</a>, <a href="https://d3js.org/" target="_blank">D3.js</a>, <a href="https://elm-lang.org/" target="_blank">elm</a>.

When I'm not working, I enjoy traveling and trying my hand at <a href="https://unsplash.com/@abdurahmanus" target="_blank">photography</a>, working on my pet projects, trying hard to learn more English.
